player_death v1.0

This mod is intended to replace the bones mod in minetest game as it replaces the bones cube with random changing gravemarkers.

License for source code is GNU Lesser General Public License, version 2.1
License for media is Attribution-ShareAlike 3.0 Unported (CC BY-SA 3.0)

The only dependency is default in minetest game.

Bug reports and suggestions on the forum thread on https://forum.minetest.net/index.php

Install this by putting it in your .minetest/mods folder.
